How to use v-unit.js to set viewport relative heights and widths:

  1. Add the v-unit.js plugin on the head, before your CSS files (helps preventing FOUC);
  2. Set classes like vh10 for 10% viewport height and vw35 for 35% of viewport width;
  3. There's no 3.
class="vh10 vw50"

height: 10% of the viewport

width: 50% of the viewport

class="vh25 vw35"

height: 25% of the viewport

width: 35% of the viewport

class="vh100 vw100"

height: 100% of the viewport

width: 100% of the viewport

class="vh31 vw63"

height: 31% of the viewport

width: 63% of the viewport
